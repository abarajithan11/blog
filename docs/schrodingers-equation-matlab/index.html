Solving Schrödinger's Equation in 1,2,3-D with MATLAB

Technical Projects • Feb 19, 2019

In my junior year, I picked an elective on Quantum Mechanics: Electronic Devices. The project that was worth 50% was to solve the time-independent Schrodinger's equation in 1,2,3-D numerically, using MATLAB.

I already had a fascination for modern physics. Therefore, I spent disproportionately more time in this project. I researched a lot, learnt about numerical methods and the intuition of Schrodinger's Equation.

Through this, I understood the Schrodinger's equation in two levels:

  1. It is mathematically a simple-harmonic-motion in space. I was fascinated to find that in 3-D, with a point charge, this SHM yields the s-p-d orbitals of the hydrogen atom.
  2. It basically says that wavefunctions are the eigenfunctions and the discrete energy values are the eigenvalues of the Hamiltonian operator constrained by the potential field.

Although I learnt eigen-things in Linear Algebra class, I did not have an intuition for them. Therefore, I spent few days building up a deeper intuition about them. I watched this entire series, took copious notes.

I then started programming using some references. While we were asked only to do it for either 1,2 or 3D using any method, I went an extra mile and did it for all three, using two methods.

Report

The intuition I gained from this, led me to write this explanation of basic QM, and this rant about how Linear Algebra is introduced poorly.
